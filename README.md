# MovieAPI_Project
* Project Overview
The Movie Finder App is a dynamic web application that allows users to search for movies and TV series, view detailed information, and explore content in an interactive and user-friendly interface. This project demonstrates the use of API integration, DOM manipulation, and modern JavaScript techniques.

* Purpose
The main goal of this project is to:

Practice working with public APIs
Improve skills in JavaScript (fetch, promises, array methods)
Build a responsive and interactive user interface
Implement real-world features like search, filtering, and sorting

* API Used
This project uses a public movie API such as:
"https://github.com/TelegramPlayground/Free-Movie-Series-DB-API"

* API Features:
Provides movie and TV series data
Returns data in JSON format
No authentication required
Includes details such as:
Title
Year
Rating
Poster
Description

* Features
1. Search Functionality
Users can search movies/series by name
Real-time fetching of results using the Fetch API

2. Movie Listings
Display movie cards dynamically
Each card includes:
Poster image
Title
Rating

3. Detailed View
Click on a movie to view:
Full description
Genre
Release date
Ratings

4. Filtering & Sorting
Filter movies based on:
Rating
Year
Sort results alphabetically or by rating
